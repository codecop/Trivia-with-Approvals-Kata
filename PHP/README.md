# Ugly Trivia with Approvals Kata PHP Setup

This is a [Composer](https://getcomposer.org/) project. Run `./vendor/bin/phpunit` to
run your tests.

* See [Trivia code base](https://github.com/caradojo/trivia).
* Create unit tests with [PHPUnit](https://phpunit.de/).
  * Any test source matching the pattern `*Test.php` will be executed.
* See [ApprovalTests for PHP](https://github.com/approvals/ApprovalTests.php).

## Check your Coverage

To measure your progress you should use code coverage tools:

* [PHPUnit Code Coverage Analysis](https://phpunit.readthedocs.io/en/8.5/code-coverage-analysis.html) - type `phpunit --coverage-html coverage`.
